# Flight of the Youtube Navigator

![alt tag](http://i.imgur.com/VxtMjVe.jpg)

### Motivation
This application provides a fast and easy way to explore Youtube videos.

### Built With (Server)
- [Node.JS / NPM]
- [Express JS]
- [Youtube API]

### Built With (Client)
- [Angular JS]
- [Angular Youtube Embed]
- [Karma JS]

### Installation
```sh
$ npm install
```

### Running (locally)
```sh
./serve.sh
```

### Test
```sh
./test.sh
```
