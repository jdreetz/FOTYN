# Flight of the Youtube Navigator

### Motivation
The purpose of this application is to provide a fast and fun way to explorer related videos within Youtube.

### Dependencies
- [Node.JS / NPM]

### Installation
```sh
$ npm install
```

### Running (locally)
```sh
./serve.sh
```

### Test
```sh
./test.sh
```


[Node.JS / NPM]:https://nodejs.org/
